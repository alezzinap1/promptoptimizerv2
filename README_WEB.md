# Prompt Engineer — Streamlit

## Запуск

```bash
pip install -r requirements.txt
streamlit run app/main.py
```

Открой: http://localhost:8501

## Переменные окружения

Создай `.env` в корне:

```
OPENROUTER_API_KEY=sk-or-v1-...
```

## Страницы

| Страница | Описание |
|----------|----------|
| Home | Генерация промптов |
| Сравнение | A/B сравнение |
| Библиотека | Сохранённые промпты |
| Техники | База техник |
| Настройки | Настройки приложения |
