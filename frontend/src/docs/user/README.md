# Prompt Optimizer — справка

**Prompt Optimizer** помогает проектировать промпты под конкретную задачу и целевую модель: структурирует запрос, подбирает техники из базы знаний, показывает превью спецификации (IDE), при необходимости задаёт уточняющие вопросы и поддерживает итерации до сохранения в библиотеку.

Тексты совпадают со встроенной справкой в приложении (**Справка** в меню).

## Типичный путь

```text
Сформулировать задачу на Home
    → при необходимости ответить на уточняющие вопросы
    → посмотреть превью IDE (цель, формат, ограничения)
    → «Создать промпт»
    → при желании итерировать или сохранить в библиотеку
```

Альтернатива без техник и мастера: **Простой режим** — вставили текст промпта, нажали «Улучшить».

## Пробный режим и свой ключ

| Ситуация | Что доступно |
|----------|----------------|
| Без своего ключа OpenRouter | Лимит **пробных токенов** и ограничение по **стоимости моделей** (дешёвые completion). Подробнее — вкладка **User Info** и **Настройки**. |
| Своим ключом в **Настройках** | Список моделей из OpenRouter по вашему тарифу, лимиты хоста на запросы/сессию могут всё ещё действовать (защита сервера). |

## Куда зайти за чем

| Вкладка справки | Содержание |
|-----------------|------------|
| **Онбординг** | Первые шаги после регистрации |
| **Главная и поток** | Home: степпер, IDE, итерация, доп. параметры, фокус |
| **Разделы приложения** | Сравнение, библиотека, техники, метрики, модели, workspaces, настройки, User Info |
| **Простой режим** | Улучшение готового промпта, пресеты, мета-промпт |
| **Глоссарий** | Термины |

Навигация между разделами — кнопками слева в окне **Справка**.
