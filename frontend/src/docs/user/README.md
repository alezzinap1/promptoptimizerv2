# Документация Prompt Engineer

Краткий справочник по приложению. Выберите раздел в списке слева:

1. **Онбординг** — с чего начать, основные экраны.
2. **Простой режим** — быстрое улучшение промпта одной кнопкой, пресеты и мета-промпт.
3. **Глоссарий** — термины и сокращения.

Эта же документация открывается из приложения: пункт **Справка** в шапке (`/help`).
