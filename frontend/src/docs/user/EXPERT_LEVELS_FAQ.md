# Уровни Junior / Mid / Senior / Creative

Кратко: это **профили генерации** в студии. Они меняют температуру, режим уточняющих вопросов и то, как подбираются техники промптинга.

| Уровень | Вопросы | Техники | Температура (по умолчанию) | Когда использовать |
|---------|---------|---------|----------------------------|-------------------|
| **Junior** | чаще спрашивает (до 5) | авто | ~0.45 | Нужны уточнения, осторожный стиль |
| **Mid** | да | авто | ~0.7 | Баланс по умолчанию |
| **Senior** | минимум | ручной набор сильных техник | ~0.85 | Знаете задачу, хотите прямой промпт |
| **Creative** | зависит от режима (текст / фото / скилл) | текст: частично ручной; фото/скилл: иначе | потолок **0.85** (стабильный формат `[PROMPT]`) | Нужно больше разнообразия, но без «перегрева» |

Подробнее про модели генерации (`gen_model`) и целевую модель (`target_model`): [EXPERT_LEVELS_AND_GENERATION_MODELS.md](./EXPERT_LEVELS_AND_GENERATION_MODELS.md).

**Reasoning-модели** (o-серия, R1, «thinking»): при их выборе студия может автоматически снизить температуру и убрать техники цепочки рассуждений из ручного набора — они избыточны для таких моделей.
