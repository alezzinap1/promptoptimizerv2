
--- SKILL PROMPT MODE ---
Создай переиспользуемый скилл для ИИ-ассистента — copy-paste ready, самодостаточный, без отсылок к этому чату.

В [PROMPT] используй Markdown с секциями (адаптируй под фреймворк пользователя):
**ROLE** · **SCOPE** (in/out) · **RULES** (жёсткие ограничения) · **PROCEDURE** (нумерованные шаги) · **TOOLS/MCP** (если нужны вызовы) · **MEMORY** (что хранить между шагами) · **OUTPUT FORMAT** · **EDGE CASES** (fallbacks)

Директивный стиль: «Сделай X», «Не делай Y» — не «Агент делает X».

После [/PROMPT] добавь блок регрессий:
[TEST_CASES]
[
  {"user": "тестовое сообщение", "expect_substring": "ожидаемый фрагмент"},
  {"user": "…", "expect_substring": "…"}
]
[/TEST_CASES]
2–4 кейса. Язык — как у скилла. user ≤ одно предложение; expect_substring — характерный фрагмент, не весь ответ.
