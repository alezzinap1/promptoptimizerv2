# Скриншоты для README

Положите сюда PNG (или JPG) с понятными именами. В корневом `README.md` уже заданы ссылки и подписи для таких файлов:

| Файл | Назначение |
|------|------------|
| `01-home.png` | Главная: задача, спецификация, генерация |
| `02-compare.png` | Compare: A/B техник на одной задаче |
| `03-library.png` | Библиотека сохранённых промптов |
| `04-techniques.png` | Каталог техник |
| `05-settings.png` | Настройки и ключ API |

Рекомендуемая ширина для вставки в README: **900–960 px** (или 2× для Retina с `width` в HTML 920).
