# Глоссарий

**OpenRouter** — сервис-шлюз к разным LLM по единому API. Приложение вызывает модели через OpenRouter.

**Пробный режим** — работа без вашего ключа: действует лимит токенов и ограничение по стоимости моделей (completion ≤ заданного порога за 1M токенов).

**Генерационная модель** — модель, которая **пишет или улучшает** промпт внутри приложения (не путать с целевой моделью, для которой готовится промпт).

**Целевая модель (target)** — модель, **для которой** вы потом используете сгенерированный промпт. Влияет на подбор техник и формулировок.

**Техника промпт-инжиниринга** — приём оформления инструкции (роль, few-shot, цепочка рассуждений и т.д.). В Home и Compare техники подбираются автоматически или вручную.

**Простой режим** — отдельный экран: один запрос к LLM, который **переписывает ваш промпт** по системным правилам, выбранному **пресету** и вашему **мета-промпту** из настроек.

**Мета-промпт (в настройках простого режима)** — дополнительные пожелания, которые добавляются к системному промпту улучшения (например: «всегда добавляй критерии успеха», «формат — маркированный список»).

**Workspace** — сохранённый профиль контекста: термины, правила стиля, ссылки — чтобы генерация на Home учитывала ваш проект.

**Сессия (в смысле авторизации)** — привязка браузера к аккаунту после входа; передаётся заголовком `X-Session-Id`.
