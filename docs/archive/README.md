# Архив документации

Здесь лежат **исторические и устаревшие** материалы: планы миграции, аудиты Streamlit, старые go/no-go анализы с противоречивыми таблицами, сжатый перенос продуктового брейнсторма.

**Не использовать** как описание текущего продукта или API. Актуальные материалы — в [`../current/README.md`](../current/README.md) и корневом `README.md`.

Папка перечислена в **`.cursorignore`**, чтобы индекс Cursor не подтягивала эти файлы автоматически.

## Содержимое

| Файл | Зачем хранить |
|------|----------------|
| `STREAMLIT_ARCHIVE.md` | Статус архивного UI в `app/` |
| `STREAMLIT_AUDIT.md` | Снимок фич Streamlit (до web как основного продукта) |
| `WEB_MIGRATION_PATH.md` | Исходный план миграции на FastAPI + React (задача выполнена) |
| `PRODUCTION_ANALYSIS.md` | Старый развёрнутый анализ; верхняя таблица противоречила фазам ниже — оставлен как история |
| `PRODUCT_IDEA_DEEP_DIVE.md` | Сжатая выжимка брейнсторма идей; полный многостраничный текст при необходимости — из git до коммита с реорганизацией `docs/` |
| `RUDIMENTS_AND_REMOVED_PATHS.md` | Что вычищено из кода/доков (Telegram bot, `bot/`, дубликаты справки) |
