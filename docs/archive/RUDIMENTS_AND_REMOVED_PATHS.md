# Рудименты и снятые с пути идеи

Фиксация после прочистки документации и кода (2026).

## Удалённые / несуществующие пути в репозитории

- **`bot/` и Telegram-бот** — в репозитории **нет** каталога `bot/`; отдельная поверхность под Telegram не поддерживается. Упоминания «Telegram bot» в комментариях `core/` заменены на актуальные (FastAPI + React, при необходимости Streamlit).
- **Дубликат пользовательской справки `docs/user/`** — удалён, чтобы не расходиться с единственным источником истины: `frontend/src/docs/user/*.md` (импорт в `Help.tsx`).

## Идеи из старых планов, которые не следует искать в коде

- **Унификация `core/` с `bot/core/`** — пункт из старого improvement-плана; второй дерево **отсутствует**, задача снята.
- **«Метрики только в Streamlit» / «нет Metrics UI в web»** — устарело: продуктовые события и блок метрик на **User Info** (`/user-info`, редирект с `/metrics`).

## Что остаётся осознанным архивом в коде

- **`app/` (Streamlit)** — намеренно сохранён как **архивный** UI; основной продукт — FastAPI + React. См. `STREAMLIT_ARCHIVE.md` в этой папке.
